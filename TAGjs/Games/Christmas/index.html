Christmas Adventure!

> This game is a first-person text-adventure game. To play, simply type commands into the box below. If you want to put on your coat, for example, you can type, "Put on coat." If you are having trouble doing something, try phrasing it differently. If you feel like the game isn't letting you do something that you want to, consider sending a feature suggestion to harrisonfackrell at mail.weber.edu.
